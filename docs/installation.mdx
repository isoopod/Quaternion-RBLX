# Installation

<Tabs>
    <TabItem value="Wally">
        Get the latest version from [wally.run/package/isoopod/quaternion-rblx](https://wally.run/package/isoopod/quaternion-rblx)
        ```toml
        Quaternion = "isoopod/quaternion-rblx@x.x.x"
        ```
    </TabItem>
    <TabItem value="Pesde">
        ```sh
        pesde add isoopod/quaternion_rbxl -a Quaternion
        ```
    </TabItem>
    <TabItem value="Manual">
        Download the latest .rbxm from [github.com/isoopod/Quaternion-RBLX/releases](https://github.com/isoopod/Quaternion-RBLX/releases) and import somewhere in your game.
    </TabItem>
</Tabs>
